# swebook-resumen-espanol

Este repositorio contiene el libro “INGENIERÍA DE SOFTWARE: SWEBOK desde mi comprensión y para principiantes”, una obra desarrollada como una interpretación y síntesis del Software Engineering Body of Knowledge (SWEBOK).

El propósito de este proyecto es ofrecer una versión resumida, estructurada y en español de los principales conceptos de la ingeniería de software, facilitando su comprensión para estudiantes, principiantes y personas interesadas en esta disciplina.

A diferencia del documento original, este material presenta un enfoque accesible sin perder el rigor conceptual, permitiendo comprender de manera clara las áreas fundamentales que conforman la ingeniería de software, tales como:

Ingeniería de requerimientos

Diseño de software

Construcción

Pruebas

Mantenimiento

Gestión de proyectos

Objetivo

Contribuir a la comunidad académica y tecnológica mediante la simplificación de conocimiento especializado, promoviendo una base sólida en ingeniería de software y fomentando el desarrollo de un enfoque disciplinado en la creación de soluciones tecnológicas.
